# ☕ Starbucks Landing Page Clone

Uma landing page inspirada na Starbucks, desenvolvida com foco em prática de Front-End, responsividade e construção de interfaces modernas utilizando HTML5 e CSS3.

O projeto recria uma experiência visual elegante e moderna, explorando boas práticas de estruturação de páginas web, estilização responsiva e organização de código.

---

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- Flexbox
- Responsividade
- Design Responsivo

---

## ✨ Funcionalidades
✅ Layout moderno inspirado na Starbucks
<br> ✅ Interface totalmente responsiva
<br> ✅ Estrutura organizada e semântica
<br> ✅ Seções visuais com imagens e banners
<br> ✅ Navegação simples e intuitiva
<br> ✅ Estilização moderna com CSS

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de aprimorar habilidades em desenvolvimento Front-End, especialmente em:

- Estruturação de páginas com HTML5
- Estilização moderna com CSS3
- Responsividade para diferentes dispositivos
- Organização de código
- Criação de interfaces inspiradas em projetos reais
  
Além do aprendizado técnico, o projeto também busca demonstrar atenção aos detalhes visuais e experiência do usuário.
